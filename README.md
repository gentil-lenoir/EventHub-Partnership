## 🎯 À propos

Numeric-Paper révolutionne la gestion d'événements en proposant des invitations digitales **impossibles à copier ou falsifier**. Notre technologie propriétaire génère des QR Codes uniques et temporaires, offrant un niveau de sécurité inégalé pour tout type d'événement.

### 🎪 Cibles du marché

- **Mariages** et cérémonies familiales
- **Conférences** et événements corporatifs  
- **Anniversaires** et celebrations privées
- **Lancements** de produits
- **Événements virtuels** et hybrides

### 🏆 Avantages compétitifs

- ✅ **100% Sécurisé** - QR Codes uniques et cryptés
- ✅ **0% Copie possible** - Protection anti-duplication
- ✅ **70% d'économie** vs impression papier
- ✅ **24/7 Support** - Assistance continue
- ✅ **3 invitations gratuites** par événement

## 🚀 Installation

### Prérequis
- Node.js (v16 ou plus)
- PHP (v7.4 ou plus)
- MySQL (v8.0 ou plus)

### Installation locale

```bash
# Cloner le repository
git clone https://github.com/gentil-lenoir/numeric-paper.git
cd numeric-paper

# Installer les dépendances frontend
npm install

# Installer les dépendances backend
composer install

# Configuration de l'environnement
cp .env.example .env
php artisan key:generate

# Migration de la base de données
php artisan migrate --seed

# Démarrer le serveur de développement
npm run dev
```

## 🎮 Utilisation

### Pour les Organisateurs
1. **Créer un compte** sur la plateforme
2. **Créer un événement** avec vos détails
3. **Personnaliser** les invitations
4. **Inviter** vos participants
5. **Gérer** les accès en temps réel

### Pour les Participants
1. **Recevoir** l'invitation par email/SMS
2. **Scanner** le QR Code sécurisé
3. **Accéder** à l'événement
4. **Participer** à l'événement

### Calculateur de Prix
La plateforme propose un calculateur interactif pour estimer les coûts :

```javascript
// Exemple de calcul
const guests = 150;
const tier = getPricingTier(guests);
const totalPrice = guests * tier.pricePerInvite;
```
